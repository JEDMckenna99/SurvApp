# Surv Frontend

React + TypeScript frontend for the Surv field service management platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run at http://localhost:3000

- React 18
- TypeScript
- Material-UI
- Redux Toolkit
- React Router
- Axios
- Vite

## Features

- Authentication with JWT
- Customer Management
- Job Scheduling
- Dashboard
- Responsive layout
